# PDFParser

#### 介绍
获取pdf中的文本与图片

#### 软件架构
软件架构说明
使用Spire获取图片
使用iTextSharp获取文本信息


#### 使用说明

1.  Parser工程提供转换所需的类库
2.  Viewer工程为使用实例
3.  可以参看Viewer界面
